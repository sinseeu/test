# ⭐ Stopwatch 2000

Y2K 레트로 감성의 스탑워치 웹앱입니다.

## 기능

- **시작 / 정지** — 타이머 시작 및 일시정지
- **랩 기록** — 구간 시간과 누적 시간 동시 표시
- **초기화** — 타이머 및 랩 기록 전체 리셋
- **최빠른 랩** 초록색, **가장 느린 랩** 빨간색으로 자동 표시

## 디자인

- Y2K 레트로 / 사이버펑크 스타일
- 네온 글로우 LCD 디스플레이
- 광택 버블 버튼
- 스캔라인 효과

## 사용법

`stopwatch.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다.
